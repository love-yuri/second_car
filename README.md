# 二手车管理系统

## 技术栈

前端：

- 主界面： `vue`  版本 : 3.3.4
- 路由导航: `vue-router` 版本 : 4.2.4
- 网络请求: `axios` 版本: 1.4.0
- 组件: `Element-ui plus` 版本 : 2.3.7

后端:

- `Sprint Boot` 版本 : 3.1.1
- `MyBatis` 版本: 3.0.1
- `Mysql` 版本 8.0.33



## 测试环境

- `archlinux` 
- `npm` 9.8.0
- `node` 16.20.1
- `jdk` 8



## 启动项目

```bash
git clone https://github.com/yuri2078/second_car.git
cd second_car
#后端
java -jar 
```
